기준 리포트, 체크리스트, 매매 전략, 실적 분석을 순서대로 연결합니다.

거래량, 볼린저 밴드, 이동평균선, MACD, RSI 14, DMI를 서버에서 자동 계산합니다.

차트 분석을 실행하면 가격 흐름, 거래량, 보조지표 요약 그래프가 표시됩니다.

뉴스 인박스

뉴스와 기사 링크는 먼저 독립 인박스에 저장한 뒤, 중복과 본문 품질을 확인하고 필요한 자료만 투자 논거로 승격합니다.

저작권 안전 모드: 기사 원문 본문은 저장하지 않고 제목, 링크, 짧은 메모와 자체 분석만 보관합니다.
뉴스 인박스를 조회하면 미승격 뉴스와 품질 확인 항목이 여기에 표시됩니다.

수동 LLM 연동

ChatGPT, Gemini 등 웹 채팅창은 직접 사용하고, 이 콘솔은 프롬프트 생성과 응답 저장/분석만 담당합니다.

자동 로그인, 자동 전송, 웹 채팅창 자동 수집은 하지 않습니다. 응답을 붙여넣으면 저장 데이터와 RAG 메모리에 연결됩니다.

보고 자동화

어닝 콜, 공시, GP 패키지를 정리해 모델 업데이트 노트와 LP 보고 스테이징 초안을 만듭니다.

웹사이트 주소는 서버에서 본문만 추출해 시장 요약에 반영합니다. PDF는 텍스트 추출/OCR을 시도하고, 텍스트·CSV·JSON 파일은 본문을 함께 반영합니다.
