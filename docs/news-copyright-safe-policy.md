# 뉴스/기사 저작권 안전 저장 정책

투자 리서치 OS의 뉴스 인박스는 기사 원문 본문을 저장하지 않습니다.

## 저장하는 것

- 제목
- 출처 URL
- 웹페이지 제목과 처리 상태
- 사용자가 직접 작성한 짧은 요약 또는 투자 메모
- 시스템이 생성한 짧은 분석 요약
- 본문 보강 필요 여부, URL-only 여부, 승격/보류/시장일지 후보 상태

## 저장하지 않는 것

- 언론사 기사 원문 전체
- 유료/로그인/스크립트 제한 페이지의 우회 수집 본문
- 외부 뉴스 서비스 결과의 대량 본문
- EMERiCs/CSF/KCIF 등 외부 포털의 원문 본문 또는 첨부 전문 자동 저장

## 화면 흐름

- 뉴스 인박스 필터에서 `승격 전`, `본문 보강 필요`, `URL-only`, `시장일지 후보`, `품질 확인`을 나눠 확인합니다.
- 투자 논거에 반영할 때도 링크와 짧은 메모 중심으로 저장 데이터/RAG에 승격합니다.
- 본문이 필요한 경우 원문 링크를 직접 확인한 뒤 사용자가 직접 요약한 메모를 추가합니다.
- EMERiCs/CSF 비즈니스 정보는 제목, 발행기관, 발행일, 링크, 보유/관심종목 관련성 점수만 자동 저장합니다.

## 제외

외부 뉴스 서비스의 검색 결과나 기사 본문을 자동으로 대량 수집하지 않습니다.
